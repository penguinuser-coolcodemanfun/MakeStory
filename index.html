Hi there! Here are some easy html stuff!

Radio (Circle choice answering) The code:

< input type="radio" >

Button (Button. Used for anything) The code:

< input type="button" >

Range (Useful for like ex: Color sliders) The code:

< input type="range" >

number (Useful for clock projects) The code:

< input type="number" >

text (You can use this for typing in text! ex: SignUp page) The code:

< input type="text" >

date (Use this for adjusting dates) The code:

< input type="date" >

file (Import files from your device) The code:

< input type="file" >
